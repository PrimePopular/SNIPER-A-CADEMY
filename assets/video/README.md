# Video assets

Drop your video files here with these exact names and both pages will pick
them up automatically — no code changes needed.

- `bootcamp-intro.mp4` → used as the background video on **bootcamp.html**
  (the signup page). Something short and loopable works best: 15–40 seconds,
  landscape, no essential info in the first few frames since text sits on
  top of it.
- `mentorship-preview.mp4` → used in the video preview player on
  **index.html** and **academy.html** (Mentorship & Community section).

Keep files reasonably small (a few MB, ideally under ~15MB) so pages still
load fast — compress with Handbrake or similar if needed.

If a file is missing or fails to load, the page automatically falls back to
a plain gradient background — nothing breaks, it just won't show video yet.
